# ProgTech

Egy olyan alkalmazás készítése, melyben lehetőségük van egyedileg összerakott, vagy előre össze állított számítógépet rendelni.
A program logolja a bizonyos információkat, a logolásra a Singleton pattern-t használom fel.
Az alap számítógép mellé köthetünk különböző biztosításokat, vagy esetleg extrákat rendelhetünk bele, ezt a Díszítő pattern felhasználásával valósítom meg.
Egy Observer figyeli, hogy van-e szükség extrára.
